# Backend Server

## To Start the Server

1. Navigate to the project folder:
```bash
cd server
```
2. Run the server:
  - For automatic reload on changes:
```bash
nodemon index.js
```
  - Or normally:
```bash
node index.js
```
The server will run on http://localhost:3000.
